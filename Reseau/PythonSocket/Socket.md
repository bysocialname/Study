# TP Reseau L2

Programmer et tester des sockets pour une communication Client-Serveur en Python.

## Communication entre client/Serveur

* Le serveur écoute sur un port spécifique et accepte les connexions des clients.
* Le client se connecte au serveur, lui envoie un message, et reçoit une réponse. 

## Exécution du programme Socket

Pour executer le code Socket, suivre les étapes ci-dessous :

```bash
Enregistrer le code du serveur sous server.py
Enregister le code du client sous client.py
Ouvrer deux terminaux.
Dans le premier terminal, exécuter le serveur : python3 server.py
Dans le deuxième terminal, exécuter le client : python3 client.py
Vous verrez les messages échangés dans les terminaux.
```
