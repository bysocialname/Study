## Gestion d'une bibliothèque

Créez un système de gestion de bibliothèque avec les fonctionnaités suivantes :

```bash
Un tableau de livres
Fonction pour ajouter un livre
Fonction pour emprunter un livre
Fonction pour retourner un livre
Fonction pour rechercher des livres par auteur
Fonction pour lister les livres disponibles
```

PS : Cette exercice est ouverte à toute contribution pour améliorer le code.
