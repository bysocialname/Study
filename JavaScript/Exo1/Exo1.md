## Page Personnelle

Creez une page HTML qui :

```bash
Affiche un message de bienvenue personnalise
Affiche l'heure actuelle dans la console
Ajoute un element au DOM qui affiche "Bienvenue [nom]"
```

PS : Ici, le DOM marche avec integration du code JS dans le fichier HTML.
Contribuer si vous trouver un autre altrnative qui marche.
