## Appartenance machine

Créer un programme qui permet de savoir les machines à un etudiant

### Algorithme

Chaque machine doit etre associé à l´étudiant proprietaire avec un code permanent et doit etre caracterisé par :

```bash
Un adresse IP
Un adresse reseau
Un code permanent
Et la salle où il se trouve
```

Et aussi, chaque etudiant doit etre caracterise par :

```bash
Un nom
Un prenom
Et un code permanent
```
