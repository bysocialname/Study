## Vente aux enchères

Ecrire un programme qui permet a deux utilisateurs qui sont le client et le huissier de justice
de se connecter à un interface specifique :

```bash
Interface Client avec le code V12026
Interface Administrateur avec le code AD2026
```

### Fonctionnalités

Le client peut demander la liste des voitures qui sont disponibles au encheres et proposer un prix sur la voiture qu´il veut encherir

Le huissier de Justice (admin) peut ajouter un voiture dans la listes des voitures, consulter la liste des voitures disponibles hors ceux deja valider et aussi valider une proposition s´il veut.
