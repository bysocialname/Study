## Adressage reseau

Créer une application où vhaque etudiant peut ajouter un ou plusieurs machines,
les supprimer ou les lister.

### Algorithme

Chaque machine est caracterisée par :

```bash
Nom du proprietaire
La salle où il est installer
Son adresse IP
Sa classe adressage
Son passerelle
Et par son etat qui peut allumer ou eteint
```
