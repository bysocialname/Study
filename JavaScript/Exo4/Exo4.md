## Calculateur d'IMC

Créez une fonction qui calcule l'Indice de Masse Corporelle (IMC) :

```bash
function calculerIMC(poids, taille) {
    Formule IMC = poids / (taille * taille)
    Retourne l'IMC et la catégorie
}
```
