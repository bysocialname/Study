## Exercice 8

On cherche à effectuer une permutation circulaire de trois nombres a, b, c

```bash
a' = b, b' = c, c' = a
```

On cherche à implenter cette fonction de façon à ce que l'action de cette fonction modifie les valeurs des variables a, b, c du programme. La suntaxe de la fonction à utiliser est :

```bash
permutation(double * pa, double * pb, double * pc)
```
