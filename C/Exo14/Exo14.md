## Exercice 14

Soit une liste de N entiers naturels (N > 2)
Ecrire le programme suivants :

```bash
1. Recherche du minimum et du maximum
2. Recherche du premier nombre dont le carré et le somme des deux precedents.
3. Recherche du nombre de sous-suites croissantes.
```

Exemple : Si la liste est la suivante : 7, 8, 1, 3, 5, 17, 15, 28, 3

```bash
1. Maximum = 28 et minimum = 1
2. C´est le nombre 3 car 3 au carré vaut 1 + 8
3. Les sous-suites sont : 7, 8 - 1, 3, 5, 17 - 15,28 - 3.
```

Donc le nombre de sous-suites est alors 4.
